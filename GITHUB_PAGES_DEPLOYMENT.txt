TROUBLESHOOTING:
================

If the app doesn't load:
1. Check GitHub Pages is enabled in repository settings
2. Verify all URLs are updated correctly
3. Check browser console for 404 errors
4. Ensure Icon.PNG file exists in public folder
5. Wait a few minutes for GitHub Pages cache to update

Common issues:
- Mixed /anwh/ and /workschedule/ URLs
- Missing Icon.PNG file
- Incorrect service worker scope
- Case sensitivity in file names
